### c_lr_necassum
- **Front:** In Logical Reasoning, what does a Necessary Assumption question ask you to find?
- **Back:** An unstated premise the argument REQUIRES to be true. If it were false, the argument would fall apart. Test it with the Negation Test: negate the answer choice — if the argument collapses, that choice is a necessary assumption.
- **Tags:** `lsat::lr::necessary_assumption`, `lsat::concept::assumption`
- **Source:** [Khan Academy — LSAT prep](https://www.khanacademy.org/prep/lsat)

### c_lr_suffassum
- **Front:** What does a Sufficient Assumption question ask for?
- **Back:** An assumption that, if added to the premises, would GUARANTEE the conclusion (make the argument valid). It need not be required — it just has to be enough to force the conclusion.
- **Tags:** `lsat::lr::sufficient_assumption`, `lsat::concept::conditional_logic`
- **Source:** [Khan Academy — LSAT prep](https://www.khanacademy.org/prep/lsat)

### c_lr_strengthen
- **Front:** Goal of a Strengthen question?
- **Back:** Pick the choice that makes the conclusion MORE likely to be true — usually by supporting a key assumption or ruling out an alternative explanation.
- **Tags:** `lsat::lr::strengthen`
- **Source:** [Khan Academy — LSAT prep](https://www.khanacademy.org/prep/lsat)

### c_lr_weaken

### c_cond_necsuff
- **Front:** In 'If A then B', which term is sufficient and which is necessary?
- **Back:** A is SUFFICIENT for B (A guarantees B). B is NECESSARY for A (A cannot happen without B). Sufficient triggers; necessary is required.
- **Tags:** `lsat::concept::conditional_logic`
- **Source:** [Wikipedia — Necessity and sufficiency](https://en.wikipedia.org/wiki/Necessity_and_sufficiency)

### c_cond_contrapositive
- **Front:** What is the contrapositive of 'If A then B', and is it valid?
- **Back:** The contrapositive is 'If not B then not A'. It is logically equivalent to the original — always valid.
- **Tags:** `lsat::concept::conditional_logic`, `lsat::concept::contrapositive`
- **Source:** [Wikipedia — Contraposition](https://en.wikipedia.org/wiki/Contraposition)

### c_cond_onlyif
- **Front:** How do you translate 'A only if B'?
- **Back:** 'A only if B' means A → B (B is necessary for A). 'Only if' introduces the NECESSARY condition.
- **Tags:** `lsat::concept::conditional_logic`
- **Source:** [forall x: Calgary (open-access logic textbook)](https://forallx.openlogicproject.org/)

### c_cond_unless
- **Front:** How do you translate 'A unless B'?
- **Back:** 'A unless B' means 'if not B, then A' (¬B → A), equivalently 'A or B'. Treat 'unless' as 'if not'.
- **Tags:** `lsat::concept::conditional_logic`
- **Source:** [forall x: Calgary (open-access logic textbook)](https://forallx.openlogicproject.org/)

### c_cond_quantifiers
- **Front:** What do 'all', 'most', and 'some' mean precisely on the LSAT?
- **Back:** All = 100%. Most = more than half (>50%). Some = at least one (≥1). 'Some' does NOT mean 'not all'.
- **Tags:** `lsat::concept::quantifiers`
- **Source:** [forall x: Calgary (open-access logic textbook)](https://forallx.openlogicproject.org/)

### p_necassum_library
- **Stimulus:** The city library should extend its weekend hours. A recent survey shows that most residents who work full-time cannot visit during the library's current weekday-evening hours.
- **Question:** The argument depends on assuming which one of the following?
  - A. Most residents of the city work full-time.
  - B. Full-time workers who cannot currently visit would use the library if weekend hours were added. **(correct)**
  - C. The library has enough funding to extend its hours.
  - D. Most libraries in other cities offer weekend hours.
  - E. No full-time worker has ever visited the library on a weekday evening.
- **Explanation:** Negation test: if those workers would NOT visit even with weekend hours, extending hours would not serve them and the recommendation collapses — so (B) is required. (A) and (D) are not needed; (C) is a separate practical issue; (E) is too extreme.
- **Tags:** `lsat::lr::necessary_assumption`, `lsat::concept::assumption`

### p_parallel_choir
- **Stimulus:** All members of the choir can read music. Jordan cannot read music. So Jordan is not a member of the choir.
- **Question:** Which one of the following arguments is most similar in its reasoning to the argument above?
  - A. All cats are mammals. Whiskers is a mammal. So Whiskers is a cat.
  - B. All doctors have degrees. Sam has no degree. So Sam is not a doctor. **(correct)**

### e_parallel_bakery
- **Stimulus:** If the bakery had received the shipment, there would be fresh bread on the shelves. There is no fresh bread on the shelves. So the bakery did not receive the shipment.
- **Question:** Which one of the following arguments is most similar in its reasoning to the argument above?
  - A. If it rains, the sidewalk gets wet. The sidewalk is wet. So it rained. _[trap: reversal]_
  - B. If the plant gets light, it grows. It did not get light. So it will not grow. _[trap: half_right]_
  - C. If the store is open, the sign is lit. The store is open. So the sign is lit. _[trap: out_of_scope]_
